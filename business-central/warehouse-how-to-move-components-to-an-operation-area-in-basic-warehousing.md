## <a name="to-move-components-to-an-operation-area-in-basic-warehouse-configurations"></a><span data-ttu-id="f6375-123">Pour déplacer les composantes vers une zone opérations dans les configurations d'entrepôt de base</span><span class="sxs-lookup"><span data-stu-id="f6375-123">To move components to an operation area in basic warehouse configurations</span></span>  
1.  <span data-ttu-id="f6375-124">Choisissez l'icône ![Ampoule qui ouvre la fonction Tell Me](media/ui-search/search_small.png "Dites-moi ce que vous voulez faire"), saisissez **Mouvement d'inventaire**, puis sélectionnez le lien correspondant.</span><span class="sxs-lookup"><span data-stu-id="f6375-124">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Inventory Movement**, and choose the relevant link.</span></span>  
2.  <span data-ttu-id="f6375-125">Sur le raccourci **Général**, renseignez le champ **N°**.</span><span class="sxs-lookup"><span data-stu-id="f6375-125">On the **General** FastTab, fill in the **No.**</span></span> <span data-ttu-id="f6375-126">.</span><span class="sxs-lookup"><span data-stu-id="f6375-126">field.</span></span> <span data-ttu-id="f6375-127">Vous pouvez appuyer sur la touche Entrée pour effectuer une sélection parmi les séries de numéros.</span><span class="sxs-lookup"><span data-stu-id="f6375-127">You can press the Enter key  to select from the number series.</span></span>  
3.  <span data-ttu-id="f6375-128">Dans le champ **Code magasin** , entrez le magasin où le mouvement a lieu.</span><span class="sxs-lookup"><span data-stu-id="f6375-128">In the **Location Code** field, enter the location where the movement occurs.</span></span>  
4.  <span data-ttu-id="f6375-129">Choisissez l'action **Extraire documents origine**.</span><span class="sxs-lookup"><span data-stu-id="f6375-129">Choose the **Get Source Documents** action.</span></span> <span data-ttu-id="f6375-130">Sinon, renseignez le champ **Document origine** , et cliquez ensuite sur le bouton **AssistEdit** dans le champ **N° origine** .</span><span class="sxs-lookup"><span data-stu-id="f6375-130">Alternatively, fill in the **Source Document** field, and then choose the **AssistEdit** button in the **Source No.** field.</span></span>  
5.  <span data-ttu-id="f6375-131">Sur la page **Documents origine** , sélectionnez l'ordre d'assemblage pour lequel vous souhaitez déplacer des composantes, puis choisissez le bouton **OK** .</span><span class="sxs-lookup"><span data-stu-id="f6375-131">On the **Source Documents** page, select the assembly order that you want to move components for, and then choose the **OK** button.</span></span>  

    <span data-ttu-id="f6375-132">Pour chaque composante nécessaire pouvant être déplacée, une ligne prélèvement et une ligne placement sont générées sur la page **Mouvements d'inventaire** .</span><span class="sxs-lookup"><span data-stu-id="f6375-132">For each needed component that can be moved, one Take line and one Place line are generated on the **Inventory Movements** page.</span></span> <span data-ttu-id="f6375-133">Tous les champs à l'exception du champ **Qté à traiter** sont préremplis en fonction des lignes document origine.</span><span class="sxs-lookup"><span data-stu-id="f6375-133">All fields except the **Qty. to Handle** field are prefilled according to the source document lines.</span></span> <span data-ttu-id="f6375-134">Le champ **Qté à traiter** est défini sur zéro jusqu'à ce que vous entriez la quantité que vous avez réellement déplacée.</span><span class="sxs-lookup"><span data-stu-id="f6375-134">The **Qty. to Handle** field is set to zero until you enter the quantity that you have actually moved.</span></span>  

    <span data-ttu-id="f6375-135">Vous pouvez modifier le code de zone de la ligne prélèvement mais uniquement en fonction de la disponibilité.</span><span class="sxs-lookup"><span data-stu-id="f6375-135">You can change the bin code on a Take line but only according to availability.</span></span> <span data-ttu-id="f6375-136">Si vous choisissez le bouton **AssistEdit** dans le champ **Code de zone** sur une ligne prélèvement, la page **Contenus de la zone** s'ouvre et affiche uniquement les zones où la composante est disponible.</span><span class="sxs-lookup"><span data-stu-id="f6375-136">If you choose the **AssistEdit** button in the **Bin Code** field on a Take line, then the **Bin Contents** page opens and only shows bins where the component is available.</span></span>  

    <span data-ttu-id="f6375-137">Vous ne pouvez pas modifier le code de zone sur une ligne Emplacement.</span><span class="sxs-lookup"><span data-stu-id="f6375-137">You cannot change the bin code on a Place line.</span></span> <span data-ttu-id="f6375-138">Seul le code de zone qui est défini sur la ligne composante du document origine est accepté.</span><span class="sxs-lookup"><span data-stu-id="f6375-138">Only the bin code that is defined on the component line of the source document is accepted.</span></span> <span data-ttu-id="f6375-139">Cela se base sur le principe que le rôle demandant une composante, qui est un ouvrier d'assemblage dans cette procédure, sait où doit être placée la composante.</span><span class="sxs-lookup"><span data-stu-id="f6375-139">This supports the principle that the role who requests a component, which is an assembly worker in this procedure, knows where the component must be placed.</span></span> <span data-ttu-id="f6375-140">Si vous souhaitez placer les composantes dans un autre zone, vous devez d'abord modifier le code de zone de la ligne composante, puis recréer les lignes mouvement d'inventaire.</span><span class="sxs-lookup"><span data-stu-id="f6375-140">If you want to place the components in a different bin, then you must first change the bin code on the component line and then re-create the inventory movement lines.</span></span>  
6.  <span data-ttu-id="f6375-141">Dans le champ **Qté à traiter**, entrez la quantité totale ou partielle que vous avez réellement déplacée.</span><span class="sxs-lookup"><span data-stu-id="f6375-141">In the **Qty. to Handle** field, enter the full or partial quantity that you have actually moved.</span></span> <span data-ttu-id="f6375-142">La valeur sur les lignes prélèvement et emplacement doit être la même.</span><span class="sxs-lookup"><span data-stu-id="f6375-142">The value on the Take and the Place lines must be the same.</span></span> <span data-ttu-id="f6375-143">Sinon, vous ne pouvez pas enregistrer le mouvement.</span><span class="sxs-lookup"><span data-stu-id="f6375-143">Otherwise, you cannot register the movement.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="f6375-144">Comme pour d'autres activités entrepôt, vous pouvez éclater la ligne d'emplacement en sélectionnant l'action **Actions**, puis en choisissant l'action **Eclater ligne**.</span><span class="sxs-lookup"><span data-stu-id="f6375-144">As in other warehouse activities, you can split the Place line by selecting the **Actions** action and then choosing the **Split Line** action.</span></span> <span data-ttu-id="f6375-145">Dans ce cas, la somme des deux lignes d'emplacement éclatées doit être égale à la quantité de la ligne prélèvement.</span><span class="sxs-lookup"><span data-stu-id="f6375-145">In that case, the sum of the two split Place lines must equal the quantity on the Take line.</span></span>  

7.  <span data-ttu-id="f6375-146">Lorsque vous êtes prêt à enregistrer les mouvements que vous avez exécutés, choisissez l'action **Enregistrer mouvement d'inventaire** .</span><span class="sxs-lookup"><span data-stu-id="f6375-146">When you are ready to register the movements that you have performed, choose the **Register Invt. Movement** action.</span></span>  

    <span data-ttu-id="f6375-147">Les écritures entrepôt sont créées, reflétant que les composants existent désormais dans les emplacements spécifiés sur les lignes d'ordre d'assemblage.</span><span class="sxs-lookup"><span data-stu-id="f6375-147">Warehouse entries are created reflecting that the components now exist in the bins specified on the assembly order lines.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="f6375-148">Contrairement au déplacement de composantes à l'aide d'un prélèvement inventaire, la consommation n'est pas reportée lorsque vous enregistrez un mouvement d'inventaire.</span><span class="sxs-lookup"><span data-stu-id="f6375-148">Unlike when you move components with an inventory pick, consumption is not posted when you register an inventory movement.</span></span> <span data-ttu-id="f6375-149">Cette étape doit être réalisée séparément en reportant la production et la consommation de l'ordre d'assemblage.</span><span class="sxs-lookup"><span data-stu-id="f6375-149">That step must be performed separately by posting the assembly order output and consumption.</span></span> <span data-ttu-id="f6375-150">Pour plus d'informations, reportez\-vous à Ordre d'assemblage.</span><span class="sxs-lookup"><span data-stu-id="f6375-150">For more information, see Assembly Order.</span></span>
